# DC Tools

DC Tools are tools for operating on the Dark Cloud series of games.

## Tools

This repo has the following tools:

| Tool | Description |
|------|-------------|
| repak | Creates a `.pak` file from a directory |

## LibDC

LibDC is a small library for Dark Cloud files.
